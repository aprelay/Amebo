## ğŸ› Troubleshooting

### Messages Not Sending

**Problem:** Message stuck or not appearing

**Solution:**
1. Check your internet connection
2. Refresh the page (pull down on mobile)
3. Log out and log back in
4. Clear browser cache

### Cannot Join Room

**Problem:** "Room not found" error

**Solution:**
1. **Double-check the room code** (case-sensitive!)
2. Ask room creator to **re-share** the code
3. Make sure code has **no extra spaces**
4. Try **copying and pasting** the code

### Messages Show as "[Failed to decrypt]"

**Problem:** Can't read messages in room

**Solution:**
1. **Wrong room code** - You need the **exact code**
2. **Room was deleted** and recreated
3. **Ask for the correct code** from room admin

### Payment Failed

**Problem:** Naira payment didn't go through

**Solution:**
1. **Check your card balance**
2. **Try different payment method** (Bank Transfer instead of Card)
3. **Contact your bank** if card is declined
4. **Wait 5 minutes** and try again
5. **Check transaction history** in app

### Crypto Balance Shows Zero

**Problem:** Wallet has funds but shows 0

**Solution:**
1. **Verify wallet address** is correct
2. **Check on blockchain explorer**:
   - Bitcoin: https://blockchain.info
   - Ethereum: https://etherscan.io
3. **Wait a few minutes** (API may be slow)
4. **Try again later** (rate limits)

### App Won't Install

**Problem:** Can't add to home screen

**Solution:**
1. **Use correct browser**:
   - iPhone/iPad: **Safari only**
   - Android: **Chrome** or **Samsung Internet**
2. **Check iOS version** (needs iOS 11.3+)
3. **Clear browser cache**
4. **Try on WiFi** instead of mobile data
